Task 04: CSS Styling & Organization

CSS Properties Applied:

  • Border: All sections have custom borders with different colors and styles
  • Margin: Spacing between elements for proper layout
  • Padding: Internal spacing within boxes for better readability
  • Float: Sidebar floats left, main content floats left for layout
  • Font-family: Multiple font families (Georgia, Arial, Verdana, Courier New)
  • Font-weight: Bold and normal weights applied throughout

Structural Elements Used:

  • IDs: #header, #footer, #task01, #task02, #task03, #task04
  • Classes: .section-box, .sidebar, .main-content, .form-container, .list-container, etc.
  • Divs: Used for creating boxes and containers throughout the page
  • Spans: Used for inline styling like .highlight, .important, .note
